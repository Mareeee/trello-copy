# Trello Copy

## App consists of

- backend
  - Express
- frontend
  - React

## Run the project

### Clone the repository

```
git clone https://github.com/Mareeee/Task-Management-Application-with-Trello-like-Design.git
cd Task-Management-Application-with-Trello-like-Design

#backend:
    cd backend
    npm install
    npm run build
    node dist/index.js

#frontend:
    npm install
    cd frontend
    npm start
```
